### [git 컨벤션] (2023. 11. 01 수정)
- feat: 새 기능 추가
- fix: 발생한 에러 수정
- style: 코드 스타일 변경 (코드 포매팅, 세미콜론 누락 등)
- refactor: 성능 개선을 위한 코드 수정
- chore: 자잘한 수정이나 빌드 업데이트
- rename: 파일 혹은 폴더명을 수정만 한 경우
- remove: 파일을 삭제만 한 경우
###### 2023. 11. 01 수정 이전에는 잘못된 컨벤션이 존재할 수 있음
